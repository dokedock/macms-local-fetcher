# 本地采集器（MacCms v10）

一个本地运行的采集工具：纯静态网页作为界面，本地服务负责请求、解析、入库、导出。

## 功能

- 支持 MacCms v10 JSON / XML / 自动识别
- 支持 SOCKS5 / SOCKS5H 代理（按源配置）
- 多源选择采集（单选/多选）、并发采集
- 任务列表：查看详情、实时进度、手动停止、删除任务记录
- 入库：SQLite（自动创建）
- 去重：按标题删重
- 导出：CSV

## 目录结构

- `collector/app/`：后端（FastAPI）
- `web/`：纯静态页面（HTML/JS/CSS）
- `data/`：运行时数据目录（自动生成，默认已被 Git 忽略）

## 运行环境

- Python 3.9+（建议 3.10/3.11）

## 安装与启动

在项目目录下执行：

```bash
python3 -m pip install -r requirements.txt
python3 -m uvicorn collector.app.main:app --host 127.0.0.1 --port 8765
```

打开页面：

```
http://127.0.0.1:8765/
```

Windows 可使用 `run.bat`，macOS 可使用 `run.command`（可能需要先在系统里允许执行）。

## 代理填写示例

- 无认证（本地代理常见）：

```
socks5h://127.0.0.1:7891
```

- 有用户名密码：

```
socks5h://user:pass@127.0.0.1:7891
```
